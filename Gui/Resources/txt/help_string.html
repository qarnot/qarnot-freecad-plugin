To start a simulation on Qarnot platform you should first create an account and then retrieve your token (from account setting). Once you have it just click on set token and fill in the dialog box.

You can now chose a solver from an analysis in the active document and set the working directory or let the field blank to use a default temporary directory. Click on start to start your simulation. The macro will first write the file and then send the computation to Qarnot platform. You can visualize current simulations states in the panel below or go to Qarnot’s console to have more details. Once a simulation is finished, select it and click load to load the results.

You can also give the simulation a name so it is easier to keep track of it and specify a working directory if needed. The results are also given a name based on the simulation’s name.
